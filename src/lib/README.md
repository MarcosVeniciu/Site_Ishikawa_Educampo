# 🛠️ Camada Lib - Validações e Utilitários

Este diretório contém a lógica de infraestrutura e segurança de dados do sistema.

## Componentes Principais

* **`schemas.ts`**: Define os contratos de dados (Zod) para toda a aplicação. 
  - Garante que os dados da fazenda obedeçam a limites zootécnicos reais.
  - Provê tipagem estática automática para o TypeScript.

Para detalhes sobre a implementação técnica das validações, consulte as DocStrings no arquivo `schemas.ts`.
